About the Free Vietnamese Dictionary Project

The official homepage of the project is http://www.informatik.uni-leipzig.de/~duc/Dict/

The Free Vietnamese Dictionary Project was started in 1997. The dictionary software is written by Ho Ngoc Duc. The databases were contributed by a number of people, including Nguyen Tien Dzung, Pham Phuong Toan, To Long Thanh, Tran Cong So, and Ho Ngoc Duc. Since 1998 the server software and the databases are developed and maintained by Ho Ngoc Duc. The JavaScript component for typing Vietnamese online was written by Tran Anh Tuan.

Currently, the following dictionaries are available:

  • English - Vietnamese dictionary (Từ điển Anh - Việt) (110.000 words)
  • Vietnamese - English dictionary (Từ điển Việt - Anh) (23.000 words)
  • French - Vietnamese dictionary (Từ điển Pháp - Việt) (48.000 words)
  • Vietnamese - French dictionary (Từ điển Việt - Pháp) (39.000 words)
  • German - Vietnamese dictionary (Từ điển Đức - Việt) (45.000 words)
  • Vietnamese - German dictionary (Từ điển Việt - Đức) (12.000 words)
  • Vietnamese dictionary (Từ điển tiếng Việt) (30.000 words)
  • Norwegian-Vietnamese dictionary (Từ điển Na Uy - Việt) (20.000 words)
Our dictionary program is extensible. It can use any dictionary database in DICT format (e.g., WordNet, Free Online Dictionary of Computing, or the free bilingual dictionaries on www.freedict.de). That is, you can download any dictionary of interest, write a 2-lines ini file to tell the program where to find the new dictionary, and run the server with that dictionary. Moreover, you can create your own dictionaries to use with our dictionary server.

You can also download the dictionary to use off-line. A version for PalmOS and PocketPC is also available.
